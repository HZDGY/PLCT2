# 一个基于JavaFX的 简单大学物理实验计算器工具
## 实现的功能:
  + 简单四则运算 + 幂运算
  + 最多5个参数自动填充
  + 最多5组×5参数 数据并行运算（不是多线程）
  + 支持 大{} 中[] 小括号() 优先级运算
  + 不依赖数据库
## 注意： ！！依赖 jdk版本 >= 17 ！！
